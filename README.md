# EIEInfo-Project
Trabajo a partir de la página EIEInfo para el proyecto final del curso IE0417


Este documento trabaja en la estructura funcional del sistema **EIE Info**; portal web de la Escuela de Ingeniería Eléctrica (UCR).

## Resumen general del proyecto

Este repositorio contiene el desarrollo y documentación del **Proyecto Final del curso IE0417 – Diseño de Software**, correspondiente a la **Semana 1**.  
Durante esta etapa se realizó un **análisis técnico y funcional completo** del sistema existente **EIE Info**, un portal web institucional construido con **Django (Python)** y desplegado mediante **Docker**.

El trabajo incluyó:
- La **documentación de la arquitectura actual**, módulos y flujos principales del sistema.  
- La elaboración de **diagramas de arquitectura, componentes y modelo de datos**.  
- Un **análisis crítico del código fuente y contenedores Docker**, identificando problemas de duplicación, acoplamiento y seguridad.  
- La definición de **oportunidades de mejora técnicas y funcionales**, con prioridad y beneficios esperados.  

El objetivo de este análisis es establecer una **base sólida para el refactor y rediseño del sistema**, aplicando buenas prácticas de arquitectura, modularidad y seguridad en las siguientes etapas del proyecto.
